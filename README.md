# x-saveclassify_viewer 説明書

--------------------------------

## 1. これは何か

* X（旧Twitter）のブックマークや気になったツイートの URL を貼り付けて、「ツイートの埋め込み表示＋自分用メモ」を一覧管理するためのシンプルなビューア。

* 保存先はローカルPCにあるブラウザ内 localStorage。JSONファイル形式で出し入れする。

* サーバー側には一切データ送信しない（オフラインでも動作可）

* 1つのブラウザにつき 10 個までタブ（分類）を持てる

* 中のプログラムはブラウザアイコンを右クリックして「メモを編集」で見ることができる。

--------------------------------

## 2. 使い方（基本）

* index.html をブラウザで開く。ファイル名ごとに別の名前のlocalStorageに紐づく。

* GitHub Pages 経由の URL でも、ローカルに保存した HTML を開いてもOK。

* https://keroppa88.github.io/x-saveclassify_viewer/

* 上記のリンクからも使える。この場合も情報の保存先はローカルPCにあるブラウザ内 localStorageとなる。

* 画面上部の「ツイートのURLをペースト」に X のツイート URL を貼り付けて「追加」ボタンを押すとツイート埋め込みが左側に、右側にメモ欄が表示される。例）https://x.com/ユーザー名/status/1234567890123456789

* 以後はブラウザを閉じても、同じブラウザ・同じPCなら一覧が残る。

--------------------------------

## 3. タブ（分類）の使い方

* 画面上部にタブが 10 個ある（デフォルト名：タブ1〜タブ10）

* タブごとに 別々のリスト が保存される（政治用、経済用…など用途分け用）

* タブ名の変更＝タブ右側の「✎」ボタンを押して表示されるダイアログに新しい名前を入力してOK を押すとその名前になる

* キーボードショートカットでタブ切り替え、0～9

--------------------------------

## 4. メモ欄とマーク

* 各ツイートの右側がメモ欄。自由にテキストを入力できる

* 下の「保存」ボタンを押すと、そのカードのメモが保存される

* メモ欄の下に ★ ● ▲ ■ の 4 ボタンがある。それぞれを押すと、メモ欄の末尾に対応する記号を挿入する。

* 「コントロール＋F」ボタンで検索しやすくするための機能。

--------------------------------

## 5. 高さ調整・順番入れ替え

* メモ欄の上にあるボタン列は上下矢印で表示位置の入れ替え。「+20・-20」ボタンで表示範囲の縦幅を拡大・縮小する。設定を変えると勝手に保存される。

--------------------------------

## 6. JSONエクスポート／インポート（バックアップ・移行）

* タブごとに現在のリストを JSON 形式で保存・読み込みできる。

--------------------------------

### 7. 削除

* 各カード右上の「削除」ボタンで、そのツイート＋メモ＋高さ設定を丸ごと削除。

--------------------------------

### 8. データの保存先について

* 保存先：ブラウザの localStorage

* ブラウザごと・PCごとに独立して保存される

* GitHub や外部サーバーには一切送信されない

* ブラウザのキャッシュやサイトデータを削除すると、このツールの保存内容も消えるので注意（バックアップは JSON エクスポートで）。

--------------------------------

### 9. 動作環境の目安

* PC用ブラウザ

* Google Chrome / Edge / Firefox の最新版を想定

* スマホブラウザでも動くが、ツイート横にメモを並べるレイアウトのため
PC での使用を前提としている

以上。
